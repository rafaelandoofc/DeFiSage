## Installation
To set up DeFiSAGE on your machine, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/DeFiSAGE.git
   cd DeFiSAGE
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the AI agent:
   ```bash
   python defisage.py
   ```
